# Horststraße Schulkalender (ICS)

Dieses Repository stellt den öffentlichen Terminkalender der KGS Horststraße als abonnierbaren ICS-Kalender bereit.
Die ICS-Datei wird regelmäßig automatisch aktualisiert und kann in Google Kalender, Outlook oder iOS-Kalender abonniert werden.

## Kalender-Link
- ICS: https://snert2k.github.io/horststrasse-ics/horststrasse.schulmanager.ics
- Startseite: https://snert2k.github.io/horststrasse-ics/

## Nutzung

### Google Kalender (Abo)
**Wichtig:** Hinzufügen per URL geht am besten am PC/Laptop.
1. https://calendar.google.com öffnen
2. Links „Weitere Kalender“ → „+“ → **Per URL**
3. ICS-Link einfügen → Hinzufügen

Einzelne Termine kann man anschließend im Termin-Menü „Duplizieren / In eigenen Kalender kopieren“.

### Outlook (Abo)
Outlook am PC:
1. Kalender → „Kalender hinzufügen“
2. „Aus dem Internet“ → ICS-Link einfügen

### iOS (iPhone/iPad)
1. Link in **Safari** öffnen
2. „Kalender abonnieren“ → Name auswählen → Fertig

## Datenquelle
Die Termine stammen aus dem eingebetteten Schulmanager-Online Kalender auf:
https://kgs-horststrasse.de/horsttermine/

## Aktualisierung
Der Kalender wird automatisch aktualisiert (geplanter GitHub-Workflow).

## Hinweis / Haftung
Ohne Gewähr. Wenn Termine im Schulkalender geändert werden, kann es je nach Kalender-App etwas dauern, bis Updates ankommen.
